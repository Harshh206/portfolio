
  # Portfolio

A personal portfolio website built with React, Vite, Tailwind CSS, and Radix UI components. This project showcases profile information, skills, projects, education, and contact links in a modern, responsive layout.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown by Vite in your browser.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Radix UI
- Lucide React icons
- TypeScript (project structure supports TSX)
